# Go Learning Journal

一个 Java 后端开发者的 Go 学习之旅。

## 动机

做了十年 Java 后端（Spring Boot + MyBatis + K8s），选择 Go 作为第二技能栈。原因：

- **K8s 原生语言** —— 日常运维云平台，Go 能让我从"用 K8s 的人"变成"懂 K8s 的人"
- **远程岗位质量高** —— Go 后端/云原生方向的远程市场比 Java 更活跃
- **学习成本低** —— 同为静态编译型语言，心智模型接近

这个仓库不以教程为目标，而是记录**一个 Java 开发者的 Go 视角**——哪些地方顺手、哪些地方别扭、Go 的设计理念跟 Java 差在哪。
